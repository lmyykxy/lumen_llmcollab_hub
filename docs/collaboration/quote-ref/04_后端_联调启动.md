## 6. 联调过程中如果发现问题

**SSE 帧时序**(常被误以为是 bug):
- `tool_call` 现在在工具开始时**立即**推(0ms),`tool_result` 在工具完成时推
- `tool_call` 和 `tool_result` 之间会有 10-100s 的真实间隔(生图/搜索),**这是预期**

**caption 兜底**:如果生图时 caption 跟过渡话重复(模型偶发),后端会清空 caption 让 image message `content=""`。前端按"4 种 content × image_url 组合"中的"纯图气泡"渲染。

**409 conversation_busy**:同 user 上一条 SSE 还没结束就再发会拒。前端 §3.5 提到的"flash 高亮"等小 issue 是预期遗留,不在联调范围。
