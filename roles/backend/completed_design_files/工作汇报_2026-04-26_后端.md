## 二、当前接口契约(简明摘要,完整版以 `docs/API.md` 为准)

**Base URL**:`http://45.76.189.48:8000`(防火墙白名单 + 5 个 IP)

| 端点 | 用途 | 关键变更 |
|---|---|---|
| `POST /users` | 创建用户 | 无变更 |
| `GET /health` | 探活 | 无变更 |
| `POST /chat` | 发消息,流式 SSE 响应 | **见 SSE 事件表** |
| `GET /users/{id}/messages` | 拉历史 | 返回的 message 现在 `content` × `image_url` 4 种组合 |
| `GET /users/{id}/subscribe` | 长连主动消息推送 | 无变更 |
| `POST /images/upload` | 上传图(≤5MB) | 无变更 |
| `GET /uploads/{id}/{file}` | 静态图 | 无变更 |

### SSE 事件契约(`POST /chat` 主流程)

```
delta*  bubble_end  [tool_call  ←→ ~10-100s ←→  tool_result  image]  done
```

| event | payload | 关键说明 |
|---|---|---|
| `delta` | `{text}` | 流式文字块 |
| `bubble_end` | `{id}` | 一个气泡定稿,id 可去重 |
| `tool_call` | `{name, args(json string)}` | **0ms 立即推**,客户端可起 indicator |
| `tool_result` | `{name, content(json string)}` | 工具完成,debug 用 |
| `image` | `{id, url, caption}` | 独立 message,**caption 可能为 ""**(兜底清空) |
| `error` | `{message}` | 紧跟 done |
| `done` | `{}` | 流终结 |

### message JSON 协议(`GET /users/{id}/messages` + Room 持久化)

```json
{
  "id": int,
  "role": "user" | "assistant",
  "content": string,
  "image_url": "/uploads/..." | null,
  "created_at": "iso8601"
}
```

`content` × `image_url` 的 4 种组合:**纯文字 / 单气泡图+文 / 纯图(兜底) / 不应出现**。

### 工具集(`/chat` 自主调用)

| 工具 | 用途 |
|---|---|
| `schedule_message` | 排定时主动消息 |
| `list_scheduled` / `cancel_scheduled` / `snooze` | 调度管理 |
| `web_search` | Kimi 嵌套调用搜外网 |
| `generate_image` | 文 / 图生图(`reference_image_url` 默认填) |

(用户上传图的 vision 预描述是**自动机制**,不是工具)
