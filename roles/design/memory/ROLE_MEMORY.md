# 设计大模型角色记忆

## 1. 设计职责

负责 Lumen / 陆小七的 UI/UX、视觉稿、交互状态、设计 token、前端交接说明。

---

## 2. 视觉气质

整体方向：

```text
深紫暗色
温暖 Material 3
樱花粉 / 酒红 / 暖光点缀
克制、有体温
不是工具 App
不是游戏数值面板
```

---

## 3. 小七视觉固定锚点

必须保持：

```text
头发
帽子
眼睛
面貌气质
体型
```

其中：

```text
眼睛是固定身份特征
眼镜是可变配饰
穿着是可变元素
```

---

## 4. 当前页面边界

已确认 MVP 主屏：

```text
Splash
Hero
Chat
Generated Image Chat
Moments
Settings
```

后续重点：

```text
日记页
图片详情页
相册页
Moments visibility
```

---

## 5. 设计禁区

```text
不要显式好感度
不要底部 Tab
不要多角色市场
不要商城式礼物页
不要任务中心
不要过度花瓣 / 白点 / 噪点
不要把小七做成普通聊天机器人头像
```
